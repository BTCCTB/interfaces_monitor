# Interfaces monitor
A php application for monitoring all interfaces.

## Requirement
* You need PHP 7.1 or newer, with session support.
* PasswordManager supports MySQL-compatible databases
    * MySQL 5.7 or newer
    * MariaDB 5.5 or newer
* LDAP PHP extension enabled
* and the [usual Symfony application requirements][1].

## Installation
execute this command to clone this repository and initialize the project:
```bash
$ git clone git@github.com:BTCCTB/interfaces_monitor.git
$ cd interfaces_monitor/
$ composer install
```
